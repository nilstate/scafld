# scafld Gate

Use the `scafld_gate` MCP tool when work is ready to be checked. It runs the independent reviewer path and returns either actionable findings or a signed receipt.

The host agent may work however it needs to work before the gate. The gate is the single accountability verb: it records the target, acceptance evidence, independent review result, and receipt status.
